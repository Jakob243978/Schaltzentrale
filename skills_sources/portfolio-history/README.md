# portfolio-history — historische Skill-Portfolio-Doku

Diese Dateien stammen aus dem am **2026-07-12 aufgelösten** `skill_dev/`-Repo
(zentrale Skill-Entwicklungs-Meta-Ebene). Sie sind **historisch** und dokumentieren
die zentrale Phase der Skill-Entwicklung (2026-05 bis 2026-07), bevor jeder Skill
zu einem self-contained SDD+PO-Projekt unter `skills_sources/<skill>/` wurde.

| Datei | Was | Status |
|---|---|---|
| `governance_log.md` | Autonome KI-Entscheidungen + Verifier-Pass-Log der zentralen Skill-Entwicklung | Historie (per Skill wird ab 2026-07-12 in `skills_sources/<skill>/docs/governance_log.md` fortgeführt) |
| `CHANGELOG.md` | Feinkörnige Skill-Änderungen der zentralen Phase | Historie (fortgeführt per Skill in `skills_sources/<skill>/CHANGELOG.md`) |
| `ROADMAP.md` | Portfolio-weite Sprint-Planung der zentralen Phase | Historie (fortgeführt per Skill in `skills_sources/<skill>/ROADMAP.md`) |

**Warum aufbewahrt statt gelöscht:** Die Governance-Historie (Entscheidungen +
Verifier-Pässe über ~60 SKILL-Tickets) ist Lern-Material und Audit-Trail. Sie wird
nicht mehr fortgeschrieben, aber nicht weggeworfen.

**Lebende Portfolio-Doku** (weiterhin aktiv, liegt eine Ebene höher unter
`skills_sources/`): `SKILLS_VISION.md` (Portfolio-Constitution) und
`skill-versions.md` (Rollback-Registry).
